How your data is stored

QuestHub uses a simple, auditable design: an append-only log on EU infrastructure.

Append-only log — no hidden mutations

  • Every quest is stored as an entry in a plain .jsonl file
  • Nothing is silently rewritten — all changes are logged entries
  • Hosted on Infomaniak VPS in Switzerland — EU-law jurisdiction
  • You can export your quests at any time (JSON or Markdown) from your room
  • Source code is AGPL-3.0 — fully auditable

What is true today

Open-source codebase (AGPL-3.0)
